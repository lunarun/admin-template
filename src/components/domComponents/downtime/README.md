# 倒计时组件

## 简介

该组件是倒计时组件，提供倒计时开始之前触发和之后触发钩子以及重置倒计时方法。

<hr>

## Attribute

| 参数 | 说明       | 类型   | 可选值                 | 默认值 |
| ---- | ---------- | ------ | ---------------------- | ------ |
| time | 倒计时时间 | Number | --                     | 60     |
| text | 倒计时文案 | String | --                     | --     |
| type | 按钮类型   | String | round / circle / plain | plain  |

<hr>

## Events

| 参数        | 说明               | 类型     | 可选值 | 默认值 |
| ----------- | ------------------ | -------- | ------ | ------ |
| beforeStart | 倒计时开始之前触发 | Function | --     | --     |
| afterEnd    | 倒计时结束之后触发 | Function | --     | --     |

## Method (通过获取组件节点进行调用)

| 参数          | 说明       | 类型 | 可选值 | 默认值 |
| ------------- | ---------- | ---- | ------ | ------ |
| resetDowntime | 重置倒计时 | --   | --     | --     |
